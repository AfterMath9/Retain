# AfterMath Apps

A collection of premium utility and social media companion applications designed for iOS and iPadOS.

---

## 1. Retain
Retain is a sophisticated social media companion for viewing and preserving public profile media.

### Key Features
* **Aggregation**: Follow public profiles and usernames in one unified feed.
* **Media Timeline**: Immersive viewers for Stories, Reels, and Explorer.
* **Permanent Likes**: Save and preserve media locally so it never expires.
* **Direct Downloads**: High-quality media archiving for offline access.

[**Download Retain in the Releases Section**](https://github.com/AfterMath9/Retain/releases)

## 2. SnapPack
SnapPack is the ultimate utility for managing and securing your Snapchat Memories locally.

### Key Features
* **Lossless Import**: Recover photos/videos from Snapchat JSON exports in original quality.
* **Privacy First**: Secure local storage with zero cloud tracking and a Biometric Locker.
* **Chronological Gallery**: Automatic organization by date with smart media validation.
* **Global Support**: Full English and Arabic (العربية) support with optimized RTL layouts.

### Quick Start
1. **Request Data**: Request your Snapchat data in **JSON** format.
2. **Extract & Import**: Download the ZIP, extract it, and select `memories_history.json` in the SnapPack Downloader tab.
3. **Download**: Hit "Start Download" to archive your memories locally.

[**Download SnapPack in the Releases Section**](https://github.com/AfterMath9/Retain/releases)
